# tykwon_wifi_scanner  
플레이스토어에 올라와져 있는 wifi분석툴들은 아래와 같은 점들에 대해서 debugging하는데 불편함이 있음.  
1. scan된 wifi가 총 몇개인지 숫자로 표시되지 않음.  
2. scan된 정보를 copy해서 저장해서 사용할 수 있게 되어져 있지 않음.  

## 테스트 환경  
Android Studio 3.2  
갤럭시 S7(Android 8.0)
